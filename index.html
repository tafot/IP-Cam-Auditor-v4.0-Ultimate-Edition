📦 Installazione

Requisiti

Python 3.8+ con pip. Consigliato ambiente virtuale.

1. Clona il repository

git clone https://github.com/tuo-username/ipcam-auditor.git
cd ipcam-auditor

2. Installa le dipendenze

pip install requests dnspython onvif-zeep opencv-python shodan aiohttp

Dipendenze opzionali ma raccomandate:

PacchettoFunzionalitàComando
requestsHTTP avanzato (sessions, proxy, auth)pip install requests
dnspythonRisoluzione DNS avanzatapip install dnspython
onvif-zeepAutenticazione e info ONVIFpip install onvif-zeep
opencv-pythonCattura frame da stream RTSPpip install opencv-python
shodanRicerca Shodan APIpip install shodan
aiohttpHTTP asincrono (performance)pip install aiohttp

3. Verifica

python ipcam_auditor.py --help

🚀 Avvio Rapido

Scenario: Scansionare la rete locale
python ipcam_auditor.py scan 192.168.1.0/24 --exploits --snapshot

Questo comando scopre tutte le telecamere nella subnet, ne identifica il brand, testa le credenziali di default, verifica vulnerabilità note e cattura screenshot.

Comandi essenziali

# Scanner rapido (solo porta 80)
python ipcam_auditor.py quick 192.168.1.0/24

# Target singolo con exploit
python ipcam_auditor.py single 203.0.113.50:80 --exploits

# Ricerca Shodan
python ipcam_auditor.py shodan "Hikvision port:80" --shodan-key YOUR_KEY

# Wizard interattivo
python ipcam_auditor.py wizard

# Ricerca FOFA
python ipcam_auditor.py fofa 'title="HIKVISION"' --fofa-email EMAIL --fofa-key KEY

Modalità di Discovery

  • scan — Scansiona un CIDR con port scanning completo
  • quick — Scansione rapida (sola porta 80)
  • single — Target singolo IP:port
  • remote — Target remoto (richiede IP:port)
  • domain — Risoluzione DNS/DDNS
  • shodan — Ricerca Shodan API
  • censys — Ricerca Censys API
  • fofa — Ricerca FOFA API
  • zoomeye — Ricerca ZoomEye API
  • dorks — Google Dork automation
  • upnp — Discovery UPnP locale
  • clouds — Scansione range IP cloud
  • list — Carica targets da file
  • wizard — Menu interattivo (default)

📊 Report Generati

Al termine della scansione, IP Cam Auditor genera report in tre formati:

📄

TXT Report

Report testuale leggibile, ideale per inclusioni in documentazione e verbali.

🌐

HTML Report

Report interattivo con statistiche, tabelle colorate, badge di severità e layout responsive.

📊

CSV Report

Formato tabellare per import in Excel, Google Sheets, SIEM o strumenti di analytics.

I report includono:

  • 📌 Riepilogo con numero di target vulnerabili/secure
  • 🔴 Sezione vulnerabile con credenziali trovate, CVE, dettagli exploit
  • 🟢 Sezione sicura (nessuna vulnerabilità trovata)
  • 🔌 Informazioni SNMP, NTLM, Telnet, snapshot catturati
  • 🏷️ Badge di severità (CRITICAL, HIGH, MEDIUM) nei report HTML
